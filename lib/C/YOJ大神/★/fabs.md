fabs() v.s. abs()
fabs 是 C 语言中的一个数学函数，用于计算一个浮点数的绝对值。fabs 函数的原型定义在 math.h 头文件中，其语法如下：
double fabs(double x);
其中，x 是一个浮点数，表示要计算绝对值的数。fabs 函数返回 x 的绝对值，即一个非负数。
需要注意的是，fabs 函数只适用于浮点数，如果要计算整数的绝对值，可以使用 abs 函数。abs 函数的原型定义在 stdlib.h 头文件中，其语法如下：
int abs(int x);
其中，x 是一个整数，表示要计算绝对值的数。abs 函数返回 x 的绝对值，即一个非负整数。
