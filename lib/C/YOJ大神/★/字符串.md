# 字符串

### 字符串的输入输出

以 #4 读入字符串并输出为例，多种方式实现字符串读取

##### 方法一 (使用字符串)

```c
char s;
scanf("%s", &s);
printf("%s\n", &s);
return 0;
```

##### 方法二 (读取至文档结束)

```c
char c;
int l=1;
while ((c = getchar()) != EOF){
    l++;
    a[l]=c;
}
```

```c
char a[1001];
int l=1;
while((a[l]=getchar())!=EOF) l++;
```

##### 方法三

```c
 char a[1000];
 scanf("%s", &a);
 i = strlen(a);
```
